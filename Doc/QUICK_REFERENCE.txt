## Thunder Client Testing

### 1. Test GET (Check Current Image)

```
GET http://localhost:8000/api/customer/1/profile-image/
Headers: Content-Type: application/json
```

Should return: `{ "profile_image": null }` or `{ "profile_image": "John_Doe_1.png" }`

## Issues & Solutions

| Issue | Check |
|-------|-------|
| No console logs | Is DevTools open? Is it Console tab? Refresh page? |
| Button doesn't respond | Are you on account.html page? Try hard refresh (Ctrl+Shift+R) |
| File dialog doesn't open | Check console for errors, screenshot and share |
| POST request fails | Check if Django is running on port 8000 |
