HOW TO TEST:
================================================================================

From your React/frontend app at http://localhost:3000, you can now use fetch:

fetch('http://127.0.0.1:8000/api/user/', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json'
  }
})
.then(response => response.json())
.then(data => console.log(data))

OR with axios:

axios.get('http://127.0.0.1:8000/api/user/')
  .then(response => console.log(response.data))
  .catch(error => console.log(error))

================================================================================
IF YOU NEED TO ADD MORE ORIGINS LATER:
================================================================================

Edit /restapi/settings.py and add to CORS_ALLOWED_ORIGINS:

CORS_ALLOWED_ORIGINS = [
    "http://localhost:3000",
    "http://127.0.0.1:3000",
    "http://example.com",  # Add more URLs here
    "https://myapp.com",
]

================================================================================
PRODUCTION NOTE:
================================================================================

For production, be more restrictive with CORS_ALLOWED_ORIGINS.
Do NOT use "*" (allow all) in production.

Instead, specify your exact frontend URL(s).
