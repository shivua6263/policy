## Plan

### 1. Update index.html with High-Quality Images
Replace all placeholder images (via.placeholder.com) with real images from Unsplash:

Each page will have:
- Consistent navbar and footer
- Page-specific content and functionality
- Responsive design
- Modern UI with cards, forms, and interactive elements

#### a) policies.html
- Display all insurance policy categories
- Health, Life, Motor, Home, Travel insurance cards
- Compare features, pricing, and benefits
- "Buy Now" buttons for each policy

#### b) agents.html
- Agent profile cards with photos
- Contact information
- Specializations
- "Consult Now" buttons
- Search/filter agents by location or specialty

#### c) claims.html
- Claim filing form
- Claim status tracking
- Required documents checklist
- FAQ section for claims

#### d) support.html
- FAQ section
- Contact form
- Live chat placeholder
- Help articles categories
- Customer service hours

#### e) quote.html
- Insurance quote calculator
- Step-by-step form
- Premium estimation
- Multiple quotes comparison
- "Get Quote" functionality

#### f) account.html
- User login/register forms
- Dashboard layout
- My Policies section
- Profile management
- Document storage
- Payment history

### 3. Update CSS (style.css)
- Add new styles for all pages
- Enhance animations and transitions
- Improve responsive design
- Add hover effects and interactive elements
- Update color scheme with professional insurance colors

### 4. Update JavaScript (script.js)
- Add page-specific functionality
- Form validation
- Dynamic content loading
- User authentication simulation
